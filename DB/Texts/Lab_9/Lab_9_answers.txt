1.	Что такое точная и неточная выборки?

Точная выборка - одна строка. Получается с помощью SELECT INTO.
Неточная выборка - Больше одной строки. Полуается с помощью обычного SELECT

2.	Объясните действие конструкций WHEN OTHERS, WHEN TO_MANY_ROWS, WHEN NO_DATA_FOUND в секции исключения. 

WHEN TO_MANY_ROWS - спицификатор секции исключений, который отслеживает исключение TO_MANY_ROWS, генерируемое, когда точная выборка (SELECT INTO) возвращает более 1 строки

WHEN NO_DATA_FOUND - спецификатор секции исключений, который отслеживает исключение NO_DATA_FOUND, генерируемое, когда неявный курсор не возвращает строк вообще

WHEN OTHERS - Отслеживает все отсальные исключения, кроме специфицированных.

3.	Объясните назначение функций SQLERRM и SQLCODE.

SQLCODE — возвращает числовой код последней ошибки в PL/SQL (целое). При отсутствии ошибки возвращает 0.

SQLERRM — возвращает текстовое сообщение (строку) последней ошибки, включающее текст Oracle-ошибки.

4.	Что такое атрибут курсора? Перечислите все атрибуты курсора и объясните их назначение.

Атрибуты курсора — предопределённые свойства курсора в PL/SQL

%ISOPEN — возвращает значение TRUE, если курсор открыт
%FOUND — определяет, найдена ли строка, удовлетворяющая условию
%NOTFOUND — возвращает TRUE, если строка не найдена
%ROWCOUNT — возвращает номер текущей строки

5.	Объясните назначение конструкции RETURNING в операторах INSERT, DELETE, UPDATE.

RETURNING в выражениях INSERT, UPDATE, DELETE используется для получения данных, измененных соответствующим выражением
Позволяет избежать дополнительного SELECT для уточнения результатов

6.	В чем отличие явного и неявного курсоров.
Явные  - объявляются разработчиком
Неявные - не тебуют объявления и OPEN, FETCH, CLOSE

8.	Что происходит по команде OPEN курсора?

OPEN открывает курсор, создавая новый результирующий набор на базе указанного запроса.

Курсор Oracle – указатель на область в PGA, в которой хранится: 
1)строки запроса, 
2)число строк, 
3)указатель на запрос в общем пуле.  
Открытие курсора – создание контекстной области PGA  – создается моментальный снимок (snapshot) данных запроса.


9.	Объясните особенность применения FOR-цикла при работе с явным курсором. 

Конструкция FOR rec IN c LOOP ... END LOOP; автоматически выполняет OPEN курсора, многократно делает FETCH и автоматически закрывает курсор по завершении. 
Внутри цикла доступна неявная запись rec (record) с полями, соответствующими столбцам запроса.

10.	Для чего применяются параметры курсора? Где они указываются?
Параметры делают курсор параметризированным: один и тот же курсорный запрос можно выполнять с разными значениями фильтров/параметров.
Это повышает переиспользуемость и позволяет динамически передавать условия.

Параметры объявляются в скобках при открытии курсора

11.	Что такое курсорная переменная? Как ее можно объявить?
Курсорные переменные - это структуры данных, которые указывают на курсорный объект

Используются для:
Передачи курсора в качестве параметра,
Чтобы отложить связь курсора с SELECT-запросом до выполнения команды OPEN

Курсорная переменная, объявленная с помощью REF CURSOR без указания RETURN может быть связана с любым запросом
Курсорная переменная, объявленная с помощью REF CURSOR с указанием RETURN может быть связана только с запросом, который возвращает результат точно соответствующий числу и типам данных в записи после фразы RETURN во время выполнения

12.	Что значит курсорный подзапрос?

Курсорный подзапрос — подзапрос, который возвращает набор строк и обычно используется в контексте курсора (например, в объявлении курсора или в конструкции OPEN ... FOR (select ...)).
Часто также имеется в виду использование подзапроса в SELECT, возвращающего набор для курсора.

13.	Объясните назначение конструкции CURRENT OF.
CURRENT OF cursor_name используется внутри UPDATE или DELETE для изменения/удаления строки, на которую в данный момент указывает активный курсор после FETCH.

14.	Для чего применяется псевдостолбец ROWID? Поясните его структуру. 

ROWID — псевдостолбец, который уникально идентифицирует физическое местоположение строки в базе данных. Это самый быстрый способ получить доступ к строке, так как ROWID напрямую
указывает на место, где находится строка.

Структура:
  -Object Number: идентифицирует таблицу или сегмент
  -Relative File Number: указывает, какой файл данных в базе содержит сегмент, указанный в Object Number
  -Block number: указание на конкретный блок данных в рамках файла, указанного в Relative File Number 
  -Row Number(Row Slot): место в блоке, в котором находится нужная строка.
  
  
 15.	Для чего применяется псевдостолбец ROWNUM?
 
 ROWNUM - псевдостолбец, возвращающий число, указывающее на порядковый номер строки в результирующем наборе. Начинается с 1 и инкрементируется на 1 по мере добавления строк в набор
