1.setTimeout()
Это встроенная функция, позволяющая выполнять какой-то callback через заданный промежуток времени (в милисекундах).
Это позволяет планировать выполнение кода без блокирования потока для других операций, что делает эту функцию удобной для управления асинхронными задачами.
Возвращает уникальный id таймаута, который можно использовать для управления таймаутом (например, в функции clearTimeout).

2.setInterval()
Это встроенная функция, позволяющая выполнять определённый callback итеративно через заданные промежутки времени (в милисекундах).
Возвращает уникальный id интервала, который потом можно использовать для упралвения интервалом (например, в функции crearInterval)

3.ref()
Это метод таймеров(timeout, interval и тд.), который используется для поддержания таймера активынм и предотвращает процесс ноды от вызода из таймера, пока таймер ещё активен.
Этот метод - часть объекта Timeout и позволяет управлять поведением таймера

4.unref()
Этот метод - противоположность ref(). Если ref() указывал на то, что процесс не может завершиться до завершения таймера, то здесь мы указываем на то, что этот-таймер это последнее событие,
которое ноде осталось обработать и тем самым разрешаем процессу ноды завершиться до завершения таймера.
