1.Создание библиотеки
ПКМ по решению->Добавить->Создать проект->Выбриаем "Статическая библиотека"->Называем->Создаём

2.Структура библиотеки
2.1 В папке с заголовочными файлами библиотеке создаём файл OS10_01.h(идентичный по названию с заголовочным файлом в первом приложении или по желанию),
после чего копируем в него содержимое из загловочного файла первого приложения
2.2Автоматически с созданием библиотеки создаётся файл Название_библиотеки.cpp. В него копируем всю реализацию функций и конструкторов, что сделали в 
первом приложении
2.3 Чтобы собрать библиотеку нажимаем Ctrl+Shift+B, после чего будет создан файл Название_библиотеки.lib(Находится в папке x64/Debug всего решения)

3.Подключение библиотеки
3.1 ПКМ по проекту, к которому хотим подключить библиотеку,после чего выбираем "Свойства"
3.2 Выбираем C/C++->Общие
3.3 В пункте "Дополнительные каталоги включаемых файлов" прописываем папку, в которой находится заголовочный файл библиотеки(OS10_1.h)
3.4 Далее выбирает Компановщик->Общее
3.5 В пункте "Дополнительные каталоги библиотек" прописываем путь к папке, где лежит файл, созданный в пункте 3.3 (Находитя в папке x64/Debug всего решения)
3.6 Далее выбираем Компановщик->Ввод
3.7 В пункте "Дополнительные зависимости" прописываем имя файла библиотеки(Название_библиотеки.lib)

4.Проверка работоспособности
4.1 Собираем проект, к которому только что подключали библиотеку
4.2 Чтобы можно было использовать функции и стурктуры из библиотеки, подключаем заголовочный файл из пункте 3.3(#include "OS10_1.h")
